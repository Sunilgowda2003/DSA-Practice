# Java Conditional Statements

This repository contains basic Java programs to understand conditional statements.

## 📚 Topics Covered
- if statement
- if-else statement
- else-if ladder
- nested if
- switch statement

## 💻 Programs Included
- IfExample.java
- IfElseExample.java
- ElseIfExample.java
- NestedIfExample.java
- WeekDays.java

## 🧠 Description
These programs demonstrate how decision-making works in Java using different conditional statements.

## 🚀 How to Run
1. Compile the program:
   javac FileName.java

2. Run the program:
   java FileName

## 🎯 Purpose
To build a strong foundation in Java programming and problem-solving.
